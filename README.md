# 🌿 Greenden — Fresh Plants & Green Living

> A clean, responsive multi-page plant e-commerce website built with pure HTML and Tailwind CSS.

---

## 📖 About the Project

**Greenden** is a beautifully designed, fully responsive plant e-commerce website that makes green living accessible to everyone. It showcases a curated catalog of premium indoor and outdoor plants, gardening accessories, and provides a seamless way for customers to get in touch — all without a backend dependency, making it easy to deploy anywhere.

Whether you're a beginner plant parent or a seasoned gardener, Greenden's clean UI and intuitive layout make browsing and buying a breeze.

---

## ✨ Features

- 🏠 **Home Page** — Hero section with call-to-action, feature highlights (Premium Quality, Fast Delivery, Expert Support), mission section, and a prominent CTA banner.
- 🛍️ **Products Page** — Grid display of 9 plant listings (Monstera Deliciosa, Snake Plant, Lucky Bamboo, Pothos Golden, Aloe Vera, Peace Lily, Rubber Plant, Spider Plant, and more) with prices and Add to Cart buttons. Includes a Plant Accessories section (pots, watering cans, soil, tools) and category filter buttons.
- 📬 **Contact Page** — Full contact form (name, email, phone, message, newsletter opt-in), business hours, address, email links, FAQ section (6 Q&As), and a store location map placeholder.
- 📱 **Mobile Responsive** — Sticky navigation with a hamburger menu, fluid grid layouts, and adaptive typography across all screen sizes.
- 🎨 **Consistent Design System** — Unified green color palette (`green-600` / `emerald`) across all pages with hover states, smooth transitions, and shadow effects.

## 🖥️ Usage

| Page | URL | Description |
|------|-----|-------------|
| Home | `index.html` | Landing page with hero, features, and mission |
| Products | `products.html` | Full plant catalog with filters and accessories |
| Contact | `contact.html` | Contact form, business hours, FAQ, and location |

Navigate between pages using the top sticky navigation bar. On mobile, tap the ☰ hamburger icon to expand the menu.

## 🧰 Tech Stack

| Technology | Purpose |
|------------|---------|
| **HTML5** | Page structure and semantic markup |
| **Tailwind CSS (CDN)** | Utility-first styling and responsive layout |
| **Vanilla JavaScript** | Mobile menu toggle and form submission handler |

No frameworks, no bundlers, no npm — just open and run.
